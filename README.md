# EstimacionTemperatura
Programa desarrollado en Matlab para predecir las temperaturas de fase de un transformador, utilizando redes neuronales.

Los datos originales no se han cargado por privacidad de la empresa.

Para ejecutar el código, debe correr directo en Matlab por secciones, procure colocar el nombre del archivo de Excel exactamente igual al que colocará en el código.
